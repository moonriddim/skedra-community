# Skedra Whiteboard Product Facts

Last updated: 2026-07-14

## What Skedra is

Skedra Whiteboard at skedra.xyz is an independent browser-based online whiteboard and infinite canvas for sketches, mind maps, flowcharts, Kanban boards, workshops and visual collaboration. The free editor can be used without registration. Skedra Cloud is an optional paid service for permanent storage and collaboration. Other products named Skedra on other domains or in app stores are separate offerings.

## Free editor

- Price: CHF 0
- Account required: No
- Storage: Local browser storage
- Editor: Infinite canvas, drawing, text, shapes, arrows, sticky notes, mind maps, flowcharts and Kanban boards
- Export formats: PNG, SVG, PDF, PPTX and portable Skedra files
- Templates and public libraries: Included

## Skedra Cloud

- Permanent cloud boards and folders
- Live multi-user collaboration
- Comments, presentations and controlled share links
- Teams, roles and access permissions
- End-to-end encrypted or server-managed encrypted boards
- Cloud storage, API keys and MCP integration

## Encryption modes

Skedra offers an end-to-end encrypted mode in which canvas content, Y.js updates, live presence data and image content are encrypted in the browser. It also offers server-managed AES-256-GCM encryption for workflows that require server-side compatibility. Operational metadata remains server-readable as described in the privacy policy.
