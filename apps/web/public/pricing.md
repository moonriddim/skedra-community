# Skedra Pricing

Last updated: 2026-07-13
Currency: CHF
Prices are per person where specified. Taxes are shown according to the Stripe checkout.

## Skedra Free

- Price: CHF 0
- Billing period: Free forever
- Account required: No
- Storage: Local browser storage
- Included: Complete infinite-canvas editor, local boards, file and image export, templates and public libraries
- Export formats: PNG, SVG, PDF, PPTX and Skedra

## Skedra Cloud Monthly

- Price: CHF 4.90 per person
- Billing period: Monthly
- Included: Everything in Free, unlimited cloud boards and folders, encrypted cloud storage, live collaboration, comments, presentations, teams, roles, share permissions, API keys and MCP
- Cancellation: Monthly

## Skedra Cloud Yearly

- Price: CHF 49 per person
- Billing period: Yearly
- Effective monthly price: Approximately CHF 4.08 per person
- Savings compared with twelve monthly payments: Approximately 17 percent
- Included: Same features as Skedra Cloud Monthly
